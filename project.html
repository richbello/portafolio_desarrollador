Tasker — CLI de automatización

Un CLI para definir tareas y pipelines con plugins simples. Pensado para productividad y reproducibilidad.

Demo Repositorio

Descripción

Tasker es una herramienta pequeña que permite definir tareas en YAML, enlazarlas y ejecutarlas de forma incremental. Soporta plugins para notificaciones, runners remotos y plantillas.

Cómo probar localmente

git clone https://github.com/youruser/tasker
cd tasker
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
tasker run --example

Características

  • Definición declarativa de tareas (YAML)
  • Plugins para entradas/salidas y notificaciones
  • Caché incremental para evitar re-ejecuciones
