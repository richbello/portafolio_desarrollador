Taller Mecánico — Sistema de Gestión y Pasarela de Pagos (Sandbox)

¿Qué problema resuelve?

Este proyecto está diseñado para talleres mecánicos que necesitan:

  • Gestionar órdenes, ventas, clientes, proveedores e inventario.
  • Auditar accesos y acciones críticas.
  • Cobrar en modo sandbox con tokenización de tarjetas.
  • Proteger credenciales y pagos con cifrado y controles operativos.

Resumen técnico y arquitectónico

Aplicación en Python con Tkinter, módulos separados, cifrado Fernet, contraseña maestra PBKDF2, licencias RSA, telemetría, y diseño configurable.

Tecnologías usadas

  • Python 3.8+, Tkinter, cryptography, Pillow, pandas, openpyxl, matplotlib

Captura de pantalla

Resultados e impacto

  • Datos cifrados, CVV no almacenado
  • Centralización operativa
  • Audit log para trazabilidad
  • Licencias firmadas para control comercial

Cómo instalar / usar

  1. Instalar dependencias con pip
  2. Colocar archivos en `C:\RICHARD\RB\2025\Taller_mecánica`
  3. Generar claves RSA y licencias
  4. Ejecutar `panel_inicio.py`
